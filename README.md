# Flashcards — приложение для изучения слов

Учебный проект, который развивается по мере изучения курса Front-End разработки.

## О проекте

Приложение для изучения иностранных слов с помощью флэшкарт.
Пользователь может добавлять карточки со словом и переводом,
листать их и отмечать выученные.

## Стек технологий

- HTML5, CSS3
- JavaScript (Vanilla)
- TypeScript
- React
- Jest (тестирование)

## Функциональность

- [ ] Просмотр карточек со словом и переводом
- [ ] Переворот карточки по клику
- [ ] Добавление новых карточек
- [ ] Удаление карточек
- [ ] Счётчик прогресса
- [ ] Тёмная тема
- [ ] Адаптив под мобильные устройства
- [ ] Подключение словарного API для автоматического перевода
- [ ] Деплой на GitHub Pages
